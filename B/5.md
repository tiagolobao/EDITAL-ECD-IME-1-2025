## 5 - Noções de intervalo de confiança e teste de hipóteses

### Intervalo de confiança

- Em uma distribuição normal (gaussiana), aproximadamente 95% dos dados estão dentro de ±1.96 desvios padrão da média. Para fins práticos, costuma-se usar ±2 desvios padrão como aproximação.

- O teorema do limite central estabelece que a distribuição das médias amostrais se aproxima de uma distribuição normal à medida que o tamanho da amostra (n) aumenta, independentemente da distribuição da população original. Para amostras grandes (geralmente n > 30), essa aproximação é considerada adequada. O desvio padrão da distribuição das médias é dada pela variância dividido por n.

- A margem de erro para um intervalo de confiança de 95% para a média populacional é calculada por: E = z * (s/√n), onde z ≈ 1.96 para 95% de confiança, s é o desvio padrão amostral e n é o tamanho da amostra. Para amostras pequenas, utiliza-se a distribuição t de Student: E = t * (s/√n).

- Interpretação correta do intervalo de confiança: Um intervalo de confiança de 95% significa que, se repetíssemos o processo de amostragem muitas vezes e construíssemos um intervalo de confiança para cada amostra, aproximadamente 95% desses intervalos conteriam a verdadeira média populacional. Não significa que há 95% de probabilidade de que a média populacional esteja dentro de um intervalo específico.

7. Uma v.a. X tem distribuição normal, com média 100 e desvio padrão 10.

(a) Qual a P(90 < X < 110)?

```
P(90 < X < 110) = P ( X+-10 )
10 = desvio padrão, portanto cerca de 68% dos dados estão dentro de um desvio padrão da média (de acordo com a tabela da distribuição normal)
```

(b) Se X' for a média de uma amostra de 16 elementos retirados dessa população, calcule P(90 < X' < 110).

```
O desvio padrão da média amostral é: 10 / sqrt(n) = 10 / 4 = 2,5
Para normalizar a distribuição para a curva Z, entao usamos Z = (X' - μ)/σ_X'

Z1 = (90 - 100) / 2,5 = -4
Z2 = (110 - 100) / 2,5 = 4
P(90 < X' < 110) = P(-4 < Z < 4)

Como Z = ±4 representa 4 desvios padrão da média, isso corresponde a praticamente 99,99% ou aproximadamente 1 (quase certeza).
```

(c) Represente, num único gráfico, as distribuições de X e X'.

```R
mu <- 100
sigma <- 10
n <- 16

sigma_x_bar <- sigma / sqrt(n)
x_vals <- seq(mu - 4*sigma, mu + 4*sigma, length.out = 1000)
y_pop <- dnorm(x_vals, mean = mu, sd = sigma)  # Distribuição de X
y_sample <- dnorm(x_vals, mean = mu, sd = sigma_x_bar)  # Distribuição de X'

plot(x_vals, y_pop, type = "l", lwd = 2, col = "blue",
     main = "Distribuições de X (população) e X' (média amostral)",
     xlab = "Valores", ylab = "Densidade",
     ylim = c(0, max(y_pop, y_sample)))
lines(x_vals, y_sample, lwd = 2, col = "red", lty = 2)
legend("topright",
       legend = c("X ~ N(100, 10)", "X' ~ N(100, 2.5)"),
       col = c("blue", "red"),
       lty = c(1, 2),
       lwd = 2)
```

```
A probabilidade de 0,95 significa que, aproximadamente a distância de 2 desvios padrões é igual a uma distancia de +-10 da média 100, portanto:

sigma = média / sqrt(n)

2 * sigma = 10
sigma = 5

5 = 10 / sqrt(n)
n = 2²
```
