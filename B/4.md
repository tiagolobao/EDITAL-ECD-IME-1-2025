4 - Noções de probabilidade

- A probabilidade vai de zero a um (0% a 100%)

- Se U e V são dois eventos mutuamente exclusivos, podemos dizer que: P(U ∪ V) = P(U) + P(V)
- Para eventos independentes: P(U ∩ V) = P(U) × P(V)

- Para um evento A, podemos afirmar que: P(A) + P(Aᶜ) = 1 (Aᶜ = complemento de A)

- O teorema de Bayes é dado por: P(A|B) = (P(B|A) × P(A)) / P(B) = P(A∩B) / P(B)

> P(A|B) Lê-se: Probabilidade de A acontecer dado que B ocorreu. Caso P(B) não seja conhecido, podemos calculá-lo com a fórmula: ∑ⱼ P(B|Aⱼ) × P(Aⱼ)

15. Considere uma urna contendo três bolas pretas e cinco bolas brancas. Retire duas bolas da urna, sem reposição. Obtenha os resultados possíveis e as respectivas probabilidades.

```python
bolas_pretas = 3
bolas_brancas = 5
total_bolas = bolas_pretas + bolas_brancas  # total = 8

# As duas pretas
p_pp = (bolas_pretas / total_bolas) * ((bolas_pretas - 1) / (total_bolas - 1))

# Uma preta e depois uma branca
p_pb = (bolas_pretas / total_bolas) * (bolas_brancas / (total_bolas - 1))

# As duas brancas
p_bb = (bolas_brancas / total_bolas) * ((bolas_brancas - 1) / (total_bolas - 1))

p_bp = (bolas_brancas / total_bolas) * (bolas_pretas / (total_bolas - 1))

print("Probabilidade de duas pretas (PP):", p_pp)
```

48. Supondo que todos os componentes do sistema da figura abaixo tenham a mesma confiabilidade p e funcionem
independentemente, obtenha a confiabilidade do sistema.


```
  A  ___o_______o_____
     |               |
  B  |               |
-----|------o--------|---
     |               |
  C  |__o________o___|

# confiabilidade p significa que p é a probabilidade de cada componente estar funcionando, então:

# Caminhos paralelos:
# - Caminho A: 2 componentes em série → P(A) = p × p = p²
# - Caminho B: 1 componente → P(B) = p
# - Caminho C: 2 componentes em série → P(C) = p × p = p²

# A confiabilidade do sistema é a probabilidade de que PELO MENOS UM dos caminhos funcione.
# Usando a regra para sistemas paralelos:
P(sistema) = 1 - [1 - P(A)] × [1 - P(B)] × [1 - P(C)]

# Substituindo:
P(sistema) = 1 - (1 - p²) × (1 - p) × (1 - p²)

# Simplificando:
P(sistema) = 1 - (1 - p²) × (1 - p) × (1 - p²)
P(sistema) = 1 - (1 - p²)² × (1 - p)

# Expandindo (1 - p²)²:
(1 - p²)² = 1 - 2p² + p⁴

# Então:
P(sistema) = 1 - (1 - 2p² + p⁴) × (1 - p)
P(sistema) = 1 - [(1 - 2p² + p⁴) - p(1 - 2p² + p⁴)]
P(sistema) = 1 - [1 - 2p² + p⁴ - p + 2p³ - p⁵]
P(sistema) = 1 - 1 + 2p² - p⁴ + p - 2p³ + p⁵
P(sistema) = p + 2p² - 2p³ - p⁴ + p⁵

# Portanto, a confiabilidade do sistema é:
P(sistema) = p⁵ - p⁴ - 2p³ + 2p² + p
```
